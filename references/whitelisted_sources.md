# Whitelisted AI/ML Research Sources

This document contains the authoritative list of trusted sources for AI/ML research curation. Always prioritize these sources when conducting research.

## Academic & Preprint Servers

### arXiv.org
- **URL Pattern**: `arxiv.org`
- **Focus**: Preprints in computer science (cs.AI, cs.LG, cs.CL, cs.CV, stat.ML)
- **Search Strategy**: Use web_search with "site:arxiv.org [topic]" or direct URL fetching
- **Citation Format**: arXiv:YYMM.NNNNN [cs.XX]

### Papers With Code
- **URL Pattern**: `paperswithcode.com`
- **Focus**: ML papers with implementation code and benchmarks
- **Value**: Links papers to code repositories and datasets

### OpenReview.net
- **URL Pattern**: `openreview.net`
- **Focus**: Peer review platform for ICLR, NeurIPS workshops, and other conferences
- **Value**: Access to reviews and discussion

## Academic Publishers & Conference Proceedings

### ACM Digital Library
- **URL Pattern**: `dl.acm.org`
- **Conferences**: NeurIPS, ICML, ICLR (older proceedings)

### IEEE Xplore
- **URL Pattern**: `ieeexplore.ieee.org`
- **Focus**: Computer vision, robotics, neural networks

### JMLR (Journal of Machine Learning Research)
- **URL Pattern**: `jmlr.org`
- **Focus**: Peer-reviewed ML research

## Technical Blogs & Company Research

### Google AI Blog
- **URL Pattern**: `ai.googleblog.com` or `blog.google/technology/ai/`
- **Focus**: Google's AI research announcements and explanations

### OpenAI Blog
- **URL Pattern**: `openai.com/blog` or `openai.com/research`
- **Focus**: OpenAI's research publications and product announcements

### Meta AI Blog
- **URL Pattern**: `ai.meta.com/blog/` or `ai.facebook.com`
- **Focus**: Meta's AI research (LLaMA, PyTorch, computer vision)

### DeepMind Blog
- **URL Pattern**: `deepmind.google/blog/` or `deepmind.com/blog`
- **Focus**: DeepMind's research (AlphaFold, game-playing AI, neuroscience)

### Anthropic Research
- **URL Pattern**: `anthropic.com/research` or `anthropic.com/news`
- **Focus**: AI safety, constitutional AI, interpretability

### Microsoft Research Blog
- **URL Pattern**: `microsoft.com/en-us/research/blog/`
- **Focus**: Microsoft's AI research across various domains

### Hugging Face Blog
- **URL Pattern**: `huggingface.co/blog`
- **Focus**: NLP, transformers, open-source models

### Distill.pub
- **URL Pattern**: `distill.pub`
- **Focus**: Clear, visual explanations of ML concepts (now archived but valuable)

## Technical Publications & Newsletters

### The Batch (DeepLearning.AI)
- **URL Pattern**: `deeplearning.ai/the-batch/`

### Import AI (Jack Clark)
- **URL Pattern**: Search for "Import AI newsletter"

### The Gradient

- **Focus**: Long-form AI research analysis and opinion

### Towards Data Science
- **URL Pattern**: `towardsdatascience.com`

### Machine Learning Mastery
- **URL Pattern**: `machinelearningmastery.com`
- **Focus**: Practical ML tutorials and guides

## Search Strategy Guidelines

### For Academic Papers
1. Start with arXiv: `site:arxiv.org [specific topic]`
2. Check Papers With Code for implementations
3. Verify with peer-reviewed sources (ACM, IEEE, JMLR) if available

### For Recent Developments
1. Company blogs: Search Google AI, OpenAI, Meta AI, Anthropic
2. Newsletters: The Batch, Import AI
3. Community: Hacker News, r/MachineLearning

### For Practical Implementation
1. Hugging Face blog for transformers/NLP
2. Towards Data Science for tutorials
3. Reddit communities for real-world experiences
4. Papers With Code for code repositories

## Citation Requirements

Every claim in the curated output must be grounded in these whitelisted sources with proper citations:

- **Format**: Use inline citations with source attribution
- **Example**: "Recent work on constitutional AI [Anthropic, 2023] demonstrates..."
- **Links**: Always include direct URLs to source material
- **Verification**: Cross-reference claims across multiple sources when possible
